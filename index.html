About this project

Este proyecto es una recreación del clásico mensaje “Sin conexión a Internet” de Google Chrome, junto con el juego del dinosaurio. Hecho con HTML, CSS y JavaScript como práctica personal para mejorar diseño y animaciones.

Tecnologías y herramientas utilizadas:

  • 💻 StackBlitz (para prototipado rápido y prueba del proyecto)
  • 🎨 Google Fonts (fuentes VT323 para estilo retro/gamer)
  • 🛠 Librería SweetAlert2 (para los modales de Game Over y dificultad)
  • 🌐 Font Awesome (para los íconos sociales de GitHub y LinkedIn)
  • 🤖 IA / ChatGPT (asistencia en lógica y diseño de funciones)
